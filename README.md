# Snake Spiel

Ein klassisches Snake-Spiel implementiert mit HTML5, CSS und JavaScript.

## Wie spielen?

1. Öffne `index.html` in einem Webbrowser
2. Klicke auf "Spiel starten"
3. Verwende die Pfeiltasten um die Schlange zu steuern:
   - ⬅️ Links
   - ⬆️ Oben  
   - ➡️ Rechts
   - ⬇️ Unten
4. Sammle das rote Futter um Punkte zu erhalten
5. Vermeide die Wände und deinen eigenen Körper

## Features

- ✅ Klassisches Snake-Gameplay
- ✅ Punktesystem mit Bestzeit-Speicherung
- ✅ Pause-Funktion
- ✅ Responsive Design für mobile Geräte
- ✅ Deutsche Benutzeroberfläche
- ✅ Restart-Funktion

## Technische Details

- **HTML5 Canvas** für das Spielfeld
- **JavaScript ES6 Klassen** für die Spiellogik
- **LocalStorage** für die Bestzeit-Speicherung
- **CSS Grid/Flexbox** für responsives Layout
